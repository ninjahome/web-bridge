Ninja 协议
ninja协议 路线图 Dapp 文档 GitHub
愿景 成为打通一切web2应用生态、web3协议及公链的通信协议
技术原理 任何人可以在Ninja上建立自己的公私钥账号体系，公钥是与他人社交互动时用户的公开身份，私钥是唯一的掌握自己社交数据所有权的凭证，通过输入其他用户的公钥地址即可与之通信，公私钥体系确保了用户可以通过Ninja协议实现真正的端对端加密通信。Ninja亦可兼容其他主流通信协议（如Nostr、Farcaster等）并可实现跨协议跨应用的通信，也支持用户输入某个公链地址（如以太坊地址、比特币地址等）找到好友进行通信。最终Ninja通过建立通信协议标准也可在服务器层面兼容目前主流的web2生态（如telegram、discord等） Tokenomic Ninja将会在不久的将来发行自己的代币，主要是用来激励早期参与者，以及在整个生态中激励所有的节点和参与者。同时在协议相关的诸多功能上以及在基于Ninja协议开发的各类应用中延展出的各类功能和场景中都会使用Ninja的代币进行结算，敬请期待。
